##### 自定义属性
| 属性| （括号内表示默认值） | 
| :-------- | :--------:|
| text| 显示文本内容（这个建议，自己在代码中去修改） |
| cdpvShowText | 是否显示 文本 |
| cdpvImage | 是否显示图片 |
| textSize | 字体大小 |
| cdpvProgress | 默认进度（0） |
| cdpvStartAngle | 开始角度（-90） |
| cdpvStrokeWidth | 边框宽度 |
| cdpvShowStroke | 是否显示边框（true） |
| cdpvStrokeColor | 边框颜色 |
| cdpvBackgroundColor | 背景颜色 |
| cdpvProgressColor | 进度颜色 |
